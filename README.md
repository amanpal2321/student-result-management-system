# 🎓 Student Result Management System

A Full Stack Web Application to manage student records and results efficiently.

## 🚀 Tech Stack
- React.js
- Spring Boot
- MySQL
- JWT Authentication

## ✨ Features
- Secure Login
- Manage Students
- View Results
- Dashboard
